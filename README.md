# All in XAU

Sitio web informativo dedicado al oro (XAU), tanto físico como en su versión crypto (XAUt, PAXG).
La idea del proyecto es una web que sea una herramienta para cualquier tipo de persona, sea comprador de oro fisico, inversor en Fondos Cotizados (ETF/ETC), Acciones Mineras, Derivados (Futuros y Opciones) o Crypto-Oro (XAUt, PAXG).
Proyecto personal en construcción, hecho por etapas como portfolio de visualización de datos.
